# Simple Adventure — klient (Vue)

## Wymagania

- **Node.js 22.x LTS** — w katalogu nadrzędnym `game/` jest [`.nvmrc`](../.nvmrc); uruchom tam `nvm install` i `nvm use`, potem wróć do `client/`.
- **npm 10+** (z Node 22).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

```
npm run build
```
